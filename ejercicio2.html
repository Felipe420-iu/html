Lista de 10 Etiquetas HTML y sus Definiciones

<html>
Define el documento como una página HTML. Es el elemento raíz de toda página web.
<head>
Contiene metadatos y enlaces a recursos externos como hojas de estilo y scripts.
<body>
Contiene todo el contenido visible de la página web, como texto, imágenes y enlaces.
<a>
Crea un enlace a otra página o recurso. Más información.
<img>
Permite mostrar imágenes en la web mediante la propiedad src para la URL de la imagen.
<p>
Define un párrafo de texto en la página.
<h1> - <h6>
Representan encabezados de distintos niveles, desde el más importante (h1) hasta el menos importante (h6).
<ul>
Crea una lista desordenada (con viñetas). Para cada elemento se usa <li>.
<ol>
Crea una lista ordenada (numerada). Cada ítem se define con <li>. Más información.
<table>
Permite crear tablas para organizar datos en filas y columnas.
